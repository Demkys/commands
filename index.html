Команды бота

Команды и триггеры

Команда / Триггер Описание
/startОтправляет приветственное сообщение с инструкциями.
/beginЗапускает отслеживание воздержания, позволяет выбрать дату начала.
/slivФиксирует рецидив, предлагает перерыв или перезапуск.
/cm, команды, нова командыСсылка на веб-страницу со списком всех команд.
/top, рангПоказывает топ пользователей по длительности воздержания.
/progress, прогрессПрогресс-бар воздержания (по умолчанию 30 дней).
/stopОстанавливает бота (только для администраторов).
/wiki, викиПоиск на Википедии с кратким описанием и изображением.
/promoАктивирует промокод для добавления NFC на баланс.
/qСоздает стикер из отвеченного сообщения.
/rules, правила, нова правилаСсылка на правила группы.
/ban, банБанит пользователя (только для администраторов).
/unban, разбанРазбанивает пользователя (только для администраторов).
/mute, мутЗамалчивает пользователя на указанный срок (только для администраторов).
/warn, нова варн, предупредитьВыдает предупреждение пользователю (только для администраторов).
/checkwarn, проверить предупрежденияПроверяет предупреждения пользователя.
сбросить предупрежденияСбрасывает все предупреждения (только для администраторов).
сбросить предупреждениеУдаляет конкретное предупреждение (только для администраторов).
нова удалить (в ответ)Удаляет ответное сообщение (только для администраторов).
закреп, закрепить (в ответ)Закрепляет сообщение (только для администраторов).
открепить, unpin (в ответ)Открепляет сообщение (только для администраторов).
бот, новаПроверяет статус и время ответа бота.
кто я, хто я, кто я?Показывает статус воздержания (дни, рецидивы, значок).
бот инфа, нова инфаДает случайный процент вероятности для запроса.
мои монетыПоказывает баланс NFC.
магазинОткрывает магазин стилей за NFC (только в лс с ботом).
мои стилиПоказывает доступные стили для выбора.
обо мнеГенерирует изображение текущего прогресса с учетом выбранного стиля.
Рекорд= (число дней)Устанавливает новый личный рекорд по количеству дней воздержания.
Рецидивы= (число рецидивов)Устанавливает новое количество зафиксированных рецидивов.
